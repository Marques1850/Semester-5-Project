Data Consent

Updated 11 November, 2023

Os dados pessoais dos utilizadores são recolhidos e tratados com o objetivo de permitir o acesso e o uso das funcionalidades da aplicação, de acordo com as preferências e necessidades de cada um. Os dados são usados para o login, para personalizar a interface, para fornecer serviços e para garantir a segurança da aplicação.

Os dados são armazenados por um período de um ano.

Os seus direitos enquanto titular de dados pessoais são garantidos pela Lei Geral de Proteção de Dados (LGPD), que entrou em vigor em 2020. Entre os seus direitos, estão:
  • Acesso aos meus dados pessoais: posso solicitar a qualquer momento uma cópia dos meus dados pessoais que estão sendo tratados por uma organização.
  • Correção dos meus dados pessoais: posso solicitar a retificação ou a atualização dos meus dados pessoais que estejam incompletos, inexatos ou desatualizados.
  • Eliminação dos meus dados pessoais: posso solicitar a exclusão dos meus dados pessoais que não sejam mais necessários para a finalidade para a qual foram coletados, ou que tenham sido tratados de forma ilegal ou abusiva.
